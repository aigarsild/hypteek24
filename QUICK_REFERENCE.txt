QUICK START COMMANDS
================================================================================

1. CREATE DATABASE (required first)
   $ mysql -u root -p
   mysql> CREATE DATABASE hypoteek;
   mysql> CREATE USER 'hypoteek'@'localhost' IDENTIFIED BY 'password';
   mysql> GRANT ALL PRIVILEGES ON hypoteek.* TO 'hypoteek'@'localhost';
   mysql> FLUSH PRIVILEGES;

2. UPDATE .env
   Edit: /Users/aigarsild/hypteek24/.env
   - Add DB_NAME, DB_USER, DB_PASSWORD
   - Add security keys from https://roots.io/salts.html

3. LINK WITH VALET
   $ cd /Users/aigarsild/hypteek24
   $ sudo valet link hypoteek
   Access: http://hypoteek.test

4. INSTALL WORDPRESS
   $ wp core install \
     --url=http://hypoteek.test \
     --title="Hypoteek" \
     --admin_user=admin \
     --admin_password=password \
     --admin_email=admin@hypoteek.test

5. ACTIVATE THEME
   $ wp theme activate hypoteek-theme

6. START DEVELOPING
   $ cd web/app/themes/hypoteek-theme
   $ npm run dev
